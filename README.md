# Adi Debel - Portfolio Website

Personal portfolio website showcasing my work as a Product Leader building enterprise AI agents and developer platforms.

## Built With

- **Next.js 15** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Vercel** - Deployment

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view.

## Deploy

Deployed on Vercel. Connected to GitHub for automatic deployments.
